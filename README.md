# Restaurant Ordering Backend API

## Overview
This project is a production-ready backend REST API built with **.NET 8** using **Clean Architecture** principles.
The goal is to demonstrate modern backend engineering practices, including proper layering, domain-driven design,
robust error handling, concurrency management, and testability.

## Tech Stack
- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- xUnit
- JWT Authentication
